Creating Tables in HTML

To create a table, use the <table> tag. Inside the <table>, you can define rows with <tr>, table headings with <th>, and table data with <td>.

Table Structure

How to do it:

  • <table> - defines the table element.
  • <tr> - defines the rows in table .
  • <th> - defines the table heading.
  • <td> - defines a table data cell (content).

Table Styling

You can also adjust your tables using CSS. For example, you can customize borders, colors, or even paddings. table { border-collapse: collapse; width: 100%; } th, td { border: 1px solid black; padding: 8px; text-align: left; } th { background-color: #f2f2f2; }
