Head Elements and HTML Entities

Head Elements

The <head> element contains the metadata about the HTML document.

Common Head Elements:

  • <title>: Gives the title.
  • <meta>: Provides metadata such as the author, charset, and viewport.
  • <link>: Used to link external resources like stylesheets.
  • <script>: Contains JavaScript or other links to an external JavaScript file.

Example of head elements:


<head>
  <title>My Webpage</title>
  <meta charset="UTF-8">
  <meta name="author" content="Your Name">
  <link rel="stylesheet" href="styles.css">
  <script type="module" src="script.js"></script>
</head>

This example shows how the <head> tag contains the followng <title>, <meta>, <link>, and <script> elements.

HTML Entities

HTML entities are used to show special characters that cant be typed onto HTML code. These characters can have special meanings in HTML (such as < for "less than") or not be on the keyboard.

Common HTML Entities:

  • &: Represents the ampersand (&)
  • <: Represents the less than sign (<)
  • >: Represents the greater than sign (>)
  • ": Represents the two quotation marks (")
  • ©: Represents the copyright symbol (Â©)

Example of using HTML entities:


<p>Use the & entity for an ampersand (&) in text.</p>
<p>The copyright symbol can be represented as &#169;</p>

This shows how to use entities to show their special characters.
