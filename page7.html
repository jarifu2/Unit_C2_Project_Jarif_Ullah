File Paths & Enhanced Text Editors

File Paths

A file path is the route used to locate a file. There are two types:

  • Absolute Path: It’s at the start of the root in the file system (e.g., C:\Users\AI\Documents\project.docx).
  • Relative Path: Relative to the current directory (e.g., Documents/project.docx).

Relative Path Examples:

  • Current folder: ./image.jpg
  • Parent folder: ../image.jpg
  • Two levels up: ../../parent/image.jpg

Relative paths make it easy to file management in a project.

Enhanced Text Editors

Enhanced text editors help coders level up their coding. Options like:

  • Notepad++ (Windows)
  • BBEdit (Mac)
  • EMACS/vi (Linux)

Integrated Development Environments (IDEs)

An IDE is a tool that combines code editing, tools, and testing in one. It’s able to support multiple programming languages.

Popular IDEs:

  • Visual Studio Code: Free and cross platform.
  • IntelliJ IDEA: Used for Java, Kotlin, and PHP.
  • PyCharm: Used for Python development.
  • WebStorm: Used for JavaScript.
  • NetBeans: Supports many languages.
