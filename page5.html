Links

The <a> tag is used to create links:

<a href="pathToFile">LinkText</a>

More link attributes:

  • Download: Lets the user download a file
  • Target: Specifies a specific place within the page where the link opens

Some common targets:

  • _blank: Opens in a new window/tab
  • _top: Opens at the top of the page
  • _self: Opens within the same iframe
  • _parent: Opens within the parent of the current frame

Example of a link with a target attribute:

<a href="URL" target="_blank">Click to open in a new tab</a>

Embedding Images

To embed an image, you can use the <img> tag with the src and alt attributes:

<img src="happyface.jpg" alt="Smiley Face">

Example of embedding an image from an external URL:

<img src="https://ex.com/hf.jpg" alt="Smiley Face">

Sizing Images

You can set the width of an image using the width attribute:

<img src="file" width="200">
