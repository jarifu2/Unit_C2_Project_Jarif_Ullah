Links

The <a> tag is used to create hyperlinks. Example syntax:

<a href="pathToFile">LinkText</a>

Common link attributes include:

  • Download: Allows users to download a file.
  • Target: Specifies where the link will open (e.g., _blank for a new tab).

Embedding Images

To embed images, use the <img> tag:

<img src="happyface.jpg" alt="Smiley Face">

You can also set dimensions:

<img src="file.jpg" width="200">

Iframes

An iframe is used to embed another HTML document or webpage within the current page. Basic syntax:

<iframe src="url" title="description"></iframe>
