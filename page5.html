Links

The <a> tag is used for the creation of hyperlinks. The syntax:

<a href="pathToFile">LinkText</a>

Common link attributes include:

  • Download: Allows users to download files.
  • Target: Makes it clear where the link will open (e.g., _blank for a new tab).

Embedding Images

To embed images use the <img> tag:

<img src="happyface.jpg" alt="Smiley Face">

You can also set the size using:

<img src="file.jpg" width="200">

Iframes

An iframe is used to embed another HTML or webpage onto a page. Basic syntax:

<iframe src="url" title="description"></iframe>

Example with dimensions:

<iframe src="https://example.com" title="Example Site" height="400" style="width: 40%;">
