Formatting Tags

<strong> - makes text bold
<html> - Root element of the HTML document.
<head> - Contains metadata about the HTML document like title, character, and author
<title> - Gives the specific title of the HTML page, which is then displayed in the browser tab.
</head>
<body> - Contains the content of the HTML document that is seen.
<h2> - Distinguishes the heading and is typically used for subheadings in a page.
<p> - Represents a paragraph.
</html> - Signifies the end of the body and HTML document.

List Elements

There are two types of list tags:

  • <ol> creates an ordered list (numbers)
  • <ul> creates an unordered list (bullets)

Both list types use <li> (list item) for the items within the lists.

<ol>, <ul>, and <li> are block-level elements.

Examples of Ordered and Unordered Lists

Ordered List:

  1. list
  2. is
  3. numbered

Unordered List:

  • list
  • has
  • bullets

Types of Body Elements

HTML has two types of body elements:

Block-level elements they automatically appear on separate lines of the page. Examples are <p> and <h1>.

Inline elements can appear next to another horizontally. Examples are <strong> and <i>.
