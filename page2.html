<!DOCTYPE html> - Defines the version of HTML being used.
<html> - The root element of the HTML document.
<head> - Contains metadata about the HTML document such as title, character encoding, etc.
<title> - Specifies the title of the HTML page, displayed in the browser tab.
</head>
<body> - Contains the visible content of the HTML document.
<h2> - Defines a heading, typically used for subheadings in the page.
<p> - Represents a paragraph of text.
</body>
</html> - Marks the end of the body content and the HTML document.
