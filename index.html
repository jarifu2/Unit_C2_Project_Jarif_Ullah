How to Embed Images

To embed an image, use the <img> tag with a src attribute to specify the file's path:

<img src="path-to-image.jpg" alt="Description of Image">

Example 1: Using a Local Image

Place the image file in the same folder as this HTML file. For example, if your image is named example.jpg, the code would look like:

<img src="example.jpg" alt="An example image">

(If there’s no local image, this won’t display anything.)

Example 2: Using an External Image URL

You can also use an external URL as the src attribute:

<img src="https://via.placeholder.com/150" alt="Placeholder Image">

Example 3: Resizing an Image

You can resize an image using the width and height attributes:

<img src="https://via.placeholder.com/150" alt="Resized Image" width="100">
