Understanding Semantic HTML

Semantic HTML uses tags to make the structure and purpose of content displayed on a webpage more clear. Also, it improves both accessibility and readability, making it easier for people who use screen readers to understand the page.

Important Semantic Elements

Page Structure

These elements distinguish the layout of a page:

  • <header>: Represents the header, contains the logo, title, or navigation links.
  • <nav>: Defines the navigation section, where menus or links are placed.
  • <main>: Skips over repetitive content like headers and footers.
  • <footer>: Represents the footer of the page, which has copyright information and additional links.

Content Organization

These elements help divide and structure content logically:

  • <article>: Used for things like blog posts or news articles.
  • <section>: Groups content like a chapter or topic section of a book.

Specific Markup

These tags make the page more interactive:

  • <aside>: Has content related to the page like a sidebar or ads.
  • <details>: Used to collapse or expand content.
  • <summary>: Gives a summary for the <details> element.
  • <mark>: Highlights the text.
  • <time>: Specific date or time.
